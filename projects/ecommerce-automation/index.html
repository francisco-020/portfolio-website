E-commerce Automation

Scripts and services that automate bulk product updates, pricing, and data synchronization across marketplaces.

  • Role:  Solo developer
  • Stack:  Python/Node.js, REST APIs, Schedulers, CSV/Sheets, Storage

Overview

Built automation pipelines to manage inventory, pricing, and variant data at scale. Jobs pull from source spreadsheets or databases, validate and transform entries, then push updates to platforms like eBay, Amazon, and Etsy.

Highlights

  • Bulk imports/updates with validation and detailed logs.
  • API integration with major marketplaces; rate-limit aware retries.
  • Scheduled jobs (cron) and on-demand triggers.
  • CSV/Google Sheets input + summary reports.
